# UGC / INFLUENCER IMAGE SKILL

**Status:** VALIDATED ✅  
**Last Updated:** January 29, 2026

Generate authentic UGC-style influencer photos — iPhone selfie quality, real environments, "girl you follow on Instagram" vibe.

---

## WHEN TO USE

- Social proof ads
- "Real person" testimonial style
- Instagram/TikTok native content
- Relatable but aspirational imagery

---

## KEY DIFFERENCE: UGC vs MODEL

| UGC/Influencer | Studio Model |
|----------------|--------------|
| iPhone selfie quality | DSLR professional |
| Real environments (bedroom, car, gym) | Studio backdrop |
| "Influencer pretty" — real but aspirational | "Model pretty" — polished |
| Casual clothes (hoodie, athleisure) | Styled outfits |
| Bright/ring light aesthetic | Directional studio lighting |
| Looking at camera, engaging | Can look away, editorial |

---

## CRITICAL ELEMENTS

### 1. MODEL — INFLUENCER PRETTY
```
Woman, mid-20s to early 30s.
GENUINELY ATTRACTIVE — the kind of girl with 50k followers.
Clear, glowing, dewy skin — healthy and radiant.
Dark hair, sleek or soft waves, well-groomed.
Minimal natural makeup — glowing base, groomed brows, subtle lip.

SHE IS PRETTY. Think popular skincare influencer.
Real but ASPIRATIONAL — someone you'd want to look like.
NOT "average person." NOT blotchy. NOT plain.
```

### 2. SKIN — GLOWING AND CLEAR
```
Dewy, healthy, radiant complexion.
Clear skin — NOT spotty, NOT blotchy.
Natural highlight on cheekbones from light.
"Glass skin" or "no makeup makeup" glow.
NOT matte flat. NOT acne. NOT dull.
```

### 3. EXPRESSION — GENUINE BIG SMILE
```
Big, genuine smile showing teeth.
Eyes engaged — smiling with whole face.
"I'm so excited to show you this" energy.
Happy, warm, inviting.
NOT awkward mid-word. NOT closed mouth. NOT forced.
```

### 4. CAMERA — PHONE SELFIE STYLE
```
iPhone front camera quality.
Bright, well-lit — ring light or window light.
Slightly above eye level angle (selfie angle).
NO PHONE IN SHOT — camera POV is the phone.
```

### 5. PRODUCT SIZE
```
Small supplement pouch — 5-6 inches tall.
SMALLER than her face.
Fits comfortably in one hand.
NOT oversized. NOT as big as her head.
```

## USAGE

1. Get product grid (Product Grid Skill)
2. Choose setting (bedroom, car, gym, etc.)
3. Fill in product description for your brand
4. Generate in Higgsfield/Nano Banana with grid as reference
5. Check:
   - Model is attractive/glowing (not plain, not blotchy)
   - Product size is correct (smaller than face)
   - Product design matches reference
   - Setting feels real/authentic
   - Expression is genuine smile
