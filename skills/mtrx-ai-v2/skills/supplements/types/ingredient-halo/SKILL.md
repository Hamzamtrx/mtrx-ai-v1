# TYPE: Ingredient Halo

## Purpose

Product-centric static with ingredients floating around in a "halo" or orbit formation. Each ingredient is visually represented and labeled. Premium, scientific, educational feel.

**Best for:** Transparency angles, skeptic audiences, education, "what's in it" messaging.

---

## Required Inputs

```
PRODUCT_NAME: [e.g., "PQ7"]
PRODUCT_DESCRIPTION: [e.g., "teal pouch with pink/green splash design"]
BRAND_NAME: [e.g., "Primal Queen"]
ACCENT_COLOR: [e.g., "pink"]
BACKGROUND: [e.g., "dark purple gradient with subtle particle bokeh"]

INGREDIENTS: [List each with visual representation]
- [Name]: [Visual] (e.g., "Berberine: Golden-brown root with berries")
- [Name]: [Visual]
- [Name]: [Visual]
...

AVATAR: [Target avatar]
H1: [Headline]
H2: [Subhead]
CTA: [Button text]
```

- [ ] Aspect ratio correct
