# 3x3 APPAREL PRODUCT GRID

## Purpose
Generate a master reference sheet showing 9 angles of the same apparel product. Use this grid as reference for all aesthetic shots to ensure product consistency.

---

## Input Requirements

**Required:**
- Full product image (shirt laid flat or on model)
- Hem tab/label close-up
- Neck label close-up

**The more reference images, the more accurate the output.**

---

## The Prompt

```
Professional apparel product photography. 3x3 grid showing ONE black t-shirt photographed 9 times.

REFERENCE IMAGES:
- Full shirt image: The product
- Hem tab close-up: Clone this exact orange tab wherever hem is visible
- Neck label close-up: Clone this exact neck tag for collar close-up frame

GRID:

ROW 1:
[1] FRONT FLAT — Shirt spread open, both sleeves out, front facing
[2] BACK FLAT — Shirt spread open, both sleeves out, back facing
[3] FOLDED — Neatly folded retail style

ROW 2:
[4] COLLAR CLOSE-UP — Clone the neck label reference image exactly
[5] HEM TAB CLOSE-UP — Clone the hem tab reference image exactly
[6] HEM WIDE — Curved hem with tab in context

ROW 3:
[7] FABRIC MACRO — Black fabric texture
[8] FOLDED ANGLE — Folded shirt from 3/4 view
[9] CASUAL — Shirt laid naturally

WHITE BACKGROUND. SOFT STUDIO LIGHT.

ONE SHIRT = IDENTICAL IN EVERY FRAME.

1:1
```

---

## Usage Notes

**Platform:** Nano Banana Pro / Higgsfield
**Aspect Ratio:** 1:1 (square)
**References:** Upload ALL close-up images alongside full product shot

**Key principle:** Less description = less hallucination. Let the reference images do the work.

---

## Grid Frame Uses

| Frame | Use For |
|-------|---------|
| [1] FRONT FLAT | Aesthetic flat lay shots - shirt spread open |
| [2] BACK FLAT | Back design shots |
| [3] FOLDED | Retail/product hero shots |
| [5] HEM TAB | Label reference for all shots |
| [6] HEM WIDE | Showing hem detail in context |
| [9] CASUAL | Lifestyle/casual aesthetic shots |

---

## Workflow

1. **Gather reference images:**
   - Full product shot
   - Hem tab close-up (clear, sharp)
   - Neck label close-up (clear, sharp)

2. **Generate 3x3 grid** using prompt above

3. **Use grid frames as reference** for aesthetic shots:
   - Reference FRONT FLAT [1] for spread-open aesthetic shots
   - Reference FOLDED [3] for retail-style shots
   - Reference HEM TAB [5] for label accuracy
